Pufferchick Trophy for CP - Version 1.0.0
A replacement sprite and name for the Stardew Hero Trophy.

:: Required Mods::
SMAPI (https://smapi.io/)
Content Patcher (https://www.nexusmods.com/stardewvalley/mods/1915)

:: What This Mod Changes::
-Changes the Stardew Hero Trophy to a cute Pufferchick shaped trophy.

:: Installation Instructions::
1. Download and install the latest version of SMAPI
2. Download and install the latest version of Content Patcher
3. Copy '[CP] Pufferchick Trophy' folder to Mods
4. You're done!
